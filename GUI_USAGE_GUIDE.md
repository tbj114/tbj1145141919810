# AxaltyX GUI 使用指南

## 概述

AxaltyX 是一个功能强大的个人学术研究统计软件，提供了完整的图形用户界面（GUI）。本指南将帮助您在有图形界面的环境中启动和使用 AxaltyX 的 GUI 功能。

## 环境要求

### 硬件要求
- 支持图形界面的计算机
- 至少 4GB 内存
- 至少 500MB 可用磁盘空间

### 软件要求
- Python 3.8 或更高版本
- PyQt6 及其依赖项
- 图形库（如 libGL）

### 依赖项安装

```bash
# 安装核心依赖
pip install pyqt6 numpy pandas matplotlib

# 安装额外依赖（用于文件格式支持）
pip install openpyxl pyreadstat chardet
```

## 启动 GUI

### 方法 1：直接运行

```bash
# 在项目根目录下运行
python -m axaltyx.main
```

### 方法 2：从Python脚本启动

```python
from axaltyx.main import main

if __name__ == "__main__":
    main()
```

## GUI 功能介绍

### 1. 主界面布局

- **顶部菜单栏**：文件、编辑、分析、图表、工具、帮助
- **左侧侧边栏**：分析菜单树，按类别组织
- **中央工作区**：数据编辑器、输出查看器、语法编辑器
- **底部状态栏**：显示当前状态和信息

### 2. 分析菜单树

侧边栏包含完整的分析菜单树，按以下类别组织：

- **描述性统计**：描述性统计、频率分析、交叉表
- **比较均值**：单样本 t 检验、独立样本 t 检验、配对样本 t 检验、单因素方差分析
- **相关分析**：相关分析、偏相关分析、距离分析
- **回归分析**：线性回归、逻辑回归、逐步回归
- **分类**：判别分析、聚类分析、因子分析
- **非参数检验**：卡方检验、 Mann-Whitney U 检验、Kruskal-Wallis H 检验
- **信度分析**： Cronbach's Alpha、因子分析
- **生存分析**： Kaplan-Meier 生存曲线、Cox 回归
- **高级分析**：路径分析、结构方程模型
- **机器学习**：决策树、随机森林、支持向量机

### 3. 分析对话框

每个分析功能都有对应的对话框，包含：

- **变量选择器**：选择要分析的变量
- **参数设置**：配置分析参数
- **统计选项**：选择要计算的统计量
- **图表选项**：选择要生成的图表类型
- **确定/取消按钮**：执行分析或取消操作

### 4. 变量选择器

变量选择器是一个通用组件，用于在分析对话框中选择变量：

- **左侧列表**：显示可用变量
- **中间按钮**：用于在列表间移动变量（>>, >, <, <<）
- **右侧列表**：显示已选择的变量
- **支持拖拽**：可以通过拖拽方式移动变量

### 5. 数据编辑器

数据编辑器允许您：
- 查看和编辑数据集
- 添加、删除、重命名变量
- 输入和修改数据
- 处理缺失值

### 6. 输出查看器

输出查看器显示分析结果：
- 表格形式的统计结果
- 生成的图表
- 分析日志和消息

### 7. 文件操作

支持的文件格式：
- **.axl**：AxaltyX 专用格式（压缩 JSON）
- **.csv**：逗号分隔值文件
- **.xlsx**：Excel 文件
- **.sav**：SPSS 文件

## 示例：使用描述性统计

1. **启动 AxaltyX**：运行 `python -m axaltyx.main`
2. **加载数据**：通过菜单栏的 "文件" -> "打开" 加载数据集
3. **打开描述性统计对话框**：在侧边栏中点击 "描述性统计" -> "描述性"
4. **选择变量**：在变量选择器中选择要分析的变量
5. **选择统计量**：勾选要计算的统计量（均值、标准差、方差等）
6. **选择图表**：勾选要生成的图表（直方图、茎叶图）
7. **执行分析**：点击 "确定" 按钮
8. **查看结果**：在输出查看器中查看分析结果和图表

## 主题和国际化

- **主题**：使用 Arco Design 风格，提供现代化的界面
- **国际化**：支持中英文切换，通过菜单栏的 "工具" -> "语言" 选择

## 故障排除

### 常见问题

1. **无法启动 GUI**
   - 检查是否安装了 PyQt6
   - 检查是否有图形界面支持
   - 检查是否安装了必要的图形库（如 libGL）

2. **分析对话框无法打开**
   - 检查是否加载了数据集
   - 检查变量是否正确定义

3. **图表无法显示**
   - 检查是否安装了 matplotlib
   - 检查数据是否有效

### 错误信息

- **libGL.so.1: cannot open shared object file**
  - 解决方案：安装图形库 `sudo apt-get install libgl1-mesa-glx`（Ubuntu/Debian）

- **PyQt6 import error**
  - 解决方案：重新安装 PyQt6 `pip install --force-reinstall pyqt6`
